- feat: add legacy WordPress URL redirects in `next.config.ts` for the upcoming Stage 6 launch. Seven 308 (permanent) rules cover (a) WP admin probes (`/wp-admin/*`, `/wp-login.php`) and (b) WP taxonomy/feed URLs Google may have indexed on the live install (`/author/*`, `/category/*`, `/tag/*`, `/feed`, `/feed/*`, `/comments/feed`). - chore: add `scripts/fetch-blog-posts.mjs` — one-shot Node fetcher that pulls each of the 6 live blog posts from home2host.com and writes two files per post under `docs/inventory/raw/blog-posts/`: a `.md` with frontmatter (title, slug, publish date, featured-image URL, excerpt, original URL) and an `.html` with the article body wrapped in a minimal stylesheet. The cleaner strips the duplicate entry-header (theme renders the title + featured image inside the body too) and removes Elementor share/related/nav widgets so the paste source is just the prose.
